📐 Architecture: Bank-First Question Loading
This tool pre-populates the Supabase question_bank table so the exam engine never calls the AI API during a live student exam.
❌ Old flow (broken)
Student starts exam → API called live per question → question displayed immediately → no verification → wrong answers + slow loading (5–8s per question)
✅ New flow (correct)
Admin (this tool): API generates → 7-layer verify → bank
Student exam: Load from bank only → instant, 100% verified, zero API latency
